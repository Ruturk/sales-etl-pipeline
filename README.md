# Sales ETL Pipeline

## Overview

This project demonstrates a simple Data Engineering ETL (Extract, Transform, Load) pipeline built using Python and Pandas.

The pipeline reads raw sales data, performs cleaning and transformations, validates data quality, and generates analytical insights.

## Architecture

Raw Sales Data (CSV)
→ Extract
→ Transform
→ Data Validation
→ Load Processed Data
→ Analytics & Visualization

## Features

* Data extraction from CSV files
* Data cleaning and transformation
* Revenue calculation
* Data quality validation
* Aggregation and analytics
* Revenue visualization using Matplotlib

## Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Sample Analytics

* Total Revenue
* Revenue by Product
* Top Performing Products
